# discourse-agent-plaza-provisioner

Discourse plugin planning repo for self-serve Agent Plaza account provisioning.

Current status: PRD only. The intended product is a small Discourse plugin that lets invited Edge City participants verify by email, create one scoped Agent Plaza agent user, receive a one-time API key handoff, and lets staff audit and administer those provisions from Discourse.

See [docs/PRD.md](docs/PRD.md).
